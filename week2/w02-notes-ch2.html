Chapter 2: Programming Basics

Chapter Summary

Comments are ignored by the program, but make your program easier to read and understand Data types are the basic building blocks of all JavaScript programs. There are six primitive data types: strings, symbols, numbers, Booleans, undefined and null. Non-primitive data types, such as arrays, functions and objects, all have a type of 'object'. Variables point to values stored in memory and are declared using the const or let keywords. Values are assigned to variables using the = operator. Strings and numbers have various properties and methods that provide information about them. Symbols are unique, immutable values. Boolean values are either true or false . There are only seven values that are false in JavaScript and these are known as 'falsy' values. Data types can be converted into other data types. Type coercion is when JavaScript tries to convert a value into another data type in order to perform an operation. Logical operators can be used to check if compound statements are true or false. Values can be compared to see if they are equal, greater than or less than other values.

Commenting Code

Commenting code is important! If will be helpful when you come back to your assignment in a few days or even a few years later. It will explain what each snippet of code does and it's purpose.

To write a comment simply enter // to comment one line of code.

Multi-line comment starts with a /* and ends in a */.

Syntax/Grammar

Javascript syntax is known as C-style syntax because of its similarities with theC programming language. It’s considered best practice to combine the two and write each statement on a new line, terminated by a semi-colon.

Example:
const message = 'Hello World!';
alert(message);

A block is a series of statements that are collected together inside curly braces

Example:
{ // this is a block containing 2 statements
const message = 'Hello!';
alert(message); }

    Declaring and Assigning Variables

    Make sure that your names for the variables describe what the variable is. Constant and variable names can start with any upper or lower-case letter, an underscore, _ , or dollar character, $ . They can also contain numbers, but cannot start with them. Remember to BE CONSISTENT.
    const = a variable that is always constant and wont be changed.
    let = a variable that may be reassigned later. (This is used instead of var now)

    Example:
    let score = 0; // This value may change during the programming

    To see the value of a variable, simply enter it in the console and press enter
    name << 'Alexa'

    You can even declare and assign multiple variables at the same time if you separate them with commas:

    Example:
    let x = 3, y = 4, z = 5;

    You should try to use const to declare most variables.

    Using const and let to declare variables means they are block scoped, so their value only exists inside the block they are declared in.

    If you assign a non-primitive value to a variable, then this is done by reference, so any changes that are subsequently made will affect all references to that object:
    const c = { value: 1 };
    let d = c; // c.value = 1, d.value = 1
    d.value = 2; // c.value = 2, d.value = 2
    In the example above, the change to the value property of d also results in the value property of c changing as well. This is because the variables c and d are both referencing the same object, so any changes to one of them will also affect the other.

    Global Scope

    Any variable declared outside of a block is said to have global scope. It is not considered good practice Learn more here. In ES6, blocks can be used to create a local scope. This means that any variables defined inside a block using the let or const will only be available inside that block and not be accessible outside of that block.

    Incrementing Values

    May use ++ or -- to increment by one either way.

    Example:
    points++; // will return 6, then increase points to 7 << 6
    ++points; // will increase points to 8, then return it << 8

    Converting Strings to Numbers

    The best way to change a string to a number is to use the Number method. This will convert the string form of a number into an actual number:
    Example:
    Number('23');
    << 23

    The preferred way of changing a number to a string is to use the String function: Example:
    String(3);
    << '3'

    Another option is to use the toString() method
    Example:
    10..toString();
    << '10'
