Chapter 3: Arrays, Logic, and Loops

Chapter Summary

This chapter covers data structures to store lists of values. They include arrays, sets and maps. This chapter also includes Logical statements that control the flow of the program, including loops.

Arrays

An array is an ordered list of values. Starts with a pair of square brackets. []
Example:
const myArray = [];
<<. []

We can find out the value of an element 0 in myArray using:
myArray[0]
This is referring to it's index.

To find the length of an array, we can use the length property:
Example:
myArray.length;
<< 6
To find the last element value in an array you will do:
avengers[avengers.length - 1];
<< 'Black Widow'
Notice that in order to get the last element you need to add -1 because the arrays start with 0. You can make an array longer by doing this:
myArray.length = 8;
If your array originally had 3 elements now it has 8, and the added elements will be undefined. You can do this same thing to make it smaller and the elements over the amount defined will be removed.

Adding Values to an Array

To assign my name to the first element of myArray we can assign it to element 0 like so:
myArray[0] = 'McKenzie';
As you can see, each element of an array can be re-assigned like a variable using the =. You can see what your array looks like by typing it in the console: myArray;

Removing Values from Arrays

You can use the delete operator. And that element will become undefined. The array will have the same number of elements, and undefined will take it's place.
Example:
delete McKenzie[0];

Destructuring Arrays

You can learn about destructuring arrays, or changing the values of the elements or changing the order by reading about it here.

Pop, Push Shift and Unshift

To remove the last item of an array you use the pop() method.
Example:
avengers.pop();

Shift removed the first element of an array:
Example:
avengers.shift();

Push method appends a new value to the end of an array:
avengers.push('Thor');

Then you can use the unshift() to add an element at the beginning of the array.

Merging Arrays

concat() method is used to merge an array with one of more arrays:
avengers.concat(['Hulk','Hawkeye',Black Widow]);
This will add these three elements to the end of the array you already have named avengers.

The join() Method

Learn about join here.

Slicing and Splicing

slice() will take a number of elements defined within the array.
Example:
avengers.slice(2,4)
This will grab the elements starting at the third item and finishes at the 4th (The item with index 4 isn't included.)

splice() removes item from an array and inserts new ones in their place
Example:
avengers.splice(3, 1, 'Scarlet Witch');
<< ['Hulk']
The first number in the parentheses tells us the index at which to start the splice. In the example we started at index 3, which is the fourth item in the array ('Hulk'). The second number tells us how many items to remove from the array. You can also do this without removing any items in the array by indicating 0 removed. Be careful with splice() as the changes are permanent.

Reverse

We can reverse the items in an array by using the reverse() method.
Example: avengers.reverse();

Sort

sort() will sort the items of the array in alphabetical order. If you have integers in your array, they will be sorted alphabetically as well. Taking them out of sequence numerically.

Sets

You want to use a set when you don't want any of the elements to be duplicated. To create a set, do: Example:
const list = new Set();

Adding Values to Sets

Values can be placed into a set using the add method:
Example:
list.add(1).add(2).add(3).add(4);
<< Set {1,2,3,4}
If you try to add a value to the set that already exist, it will be ignored.

It can also be done like so:
const numbers = new Set([1,2,3]);
All non-primitive values, such as arrays and objects, are considered unique values, even if they contain the same values. On the face of it, this appears to allow duplicate values appear in a set:
Example:
const arrays = new Set().add([1]).add([1]);
arrays
<< Set { [ 1 ], [ 1 ] }

You can find the numbers of values in a set by using the size() method.

has() method can be checked to see if a value is in a set.
Example:
avengers.has('Superman');

Using the delete() method will remove a value from the set. It will return a boolean true or false. True will mean it's removed. False will indicate it was not there in the first place and cannot be removed.

clear() method will remove all values from the set.

Converting Sets to Arrays

Learn about them here. It's also possible to use the Array.from() method to convert a set into an array. The following code would achieve the same result as using the spread operator above:
Example:
const shoppingSet = new Set().add('Apples').add('Bananas').add('Beans');
const shoppingArray = Array.from(shoppingSet);
By combining this use of the spread operator with the ability to pass an array to the new Set() constructor, we now have a convenient way to create a copy of an array with any duplicate values removed:
Example:
const duplicate = [3, 1, 4, 1, 5, 9, 2, 6 ,5,3,5,9];
<< [ 3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5, 9 ]
const nonDuplicate = [...new Set(repeatedArray)];
<< [ 3, 1, 4, 5, 9, 2, 6 ]
