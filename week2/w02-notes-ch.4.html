Arrow Functions

An arrow function can take a function written like so:
Example:
function square(x){
return x*x;
}
And be re-written like this:
Example:
const square = x => x*x;
Advantages to using arrow functions:
They are much less verbose than normal function declarations. Single parameters don't need putting into parentheses. The body of the function doesn't need placing inside a block if it's only one line. The return keyword isn't required if the return statement is the only statement in the body of the function. They don't bind their own value of this to the function (we'll see why this is a particularly useful property when we cover objects later in the book).
