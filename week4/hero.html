Hidden input fields

Hidden fields can be created using the type='hidden' It is not displayed to the browser but have a value attribute that contains information the use already provided. This is not to be used with sensitive data as it still is shown in html.

File input

Can be created by typing type='file'. These are used to upload files. Most browsers provide a browse button to search for the file needed.
